# 🛒 Retail Data Engineering Project

## 📌 Overview

End-to-end data engineering pipeline built using Azure services and Synapse Analytics.

## 🧱 Architecture

* Azure Data Lake Storage Gen2 (Data Storage)
* Azure Synapse Analytics (SQL + Spark)
* PySpark (Data transformation)

## 🔄 Pipeline Flow

Bronze → Silver → Gold

## 📊 Use Case

* Daily revenue calculation
* Top product category analysis

## ⚙️ Technologies Used

* Python (PySpark)
* SQL (Serverless SQL)
* Azure Storage
* Synapse Analytics

## 📈 Output

* External tables created on gold layer
* Queryable via Synapse SQL

## 🚀 Key Learnings

* Handling external tables in Synapse
* Managing IAM permissions
* Working with ADLS Gen2
